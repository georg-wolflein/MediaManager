Exit confirmation

This indicates whether or not the application should show an exit warning when it is closed. This can be useful if you aloways forget to save the database before closing the application.

Save button

Click here to save the settings. A confirmation will pop up with detailed instructions on how to continue. This warning will not show when this dialog was opened because an invalid database path was specified.
