Main menu

This is the main menu, providing many application functions.

Media database table

In the media database table you can see the current state of the media database. Hold Ctrl to select multiple items.

Status bar

The status bar provides basic information about the application and when the media database was last updated.

Media Items

These are the media item rows with the following information:
  • Media ID
  • Name
  • Volume number
  • Volume name
  • Number of CDs/DVDs (if it is a pack)
  • Artist
  • Location (if you have several shelves)
  • Comments
