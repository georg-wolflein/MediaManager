Print window

Print preview

This shows a preview of the document you are about to print. Use the scale below to scale the preview size.

Items to print

This shows the items that you selected earlier to print. Click on one of them for more information.

Item information

This displays the currently selected item's information. See the edit item window layout for more information about the controls.

Cancel button

Click here to abort the operation. Nothing will be printed.

Setup button

This will open a dialog about the page settings, where you can descide paper format, size, margins, etc.

Print button

Finally, click here to print the document. Another dialog will open, where you can select the printer, and then press "Print" in that dialog as well.
