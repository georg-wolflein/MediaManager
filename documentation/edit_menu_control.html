Edit menu control

Media Database In Excel...

When clicking on this element, Microsoft Office Excel will open and you can edit data in there. Please note that this only works if you already have Excel installed.
Top

Edit Selected Item...

This will open the edit item window about the currently selected item in the media database table. Please note that only one element can be edited at a time.
Top

Delete Selected Item(s)

Click here to simply delete the selected items forever. This will accept more than one selected item.
Top

Add Item...

This will show the edit item window, just that you are adding an item.
Top

Preferences...
