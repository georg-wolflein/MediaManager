Edit item window

Name

This indicates the name of the media element.

Volume number

If this piece of media belongs to a series, this will show the volume number.

Volume name

If this particular element in a series has a name, it will be indicated here.

Artist

This shows the artist.

QR code

This is a preview of the QR code that will be printed on the sticker.

Location

If you want to specify the location of this object (shelve, room, etc.), then do it here.

ID

This is the identification number of the element. Do not edit this, since it will corrupt the data!

Number of CDs

If the CDs come in a pack, they number of CDs are displayed here.

Comments

Just enter some comments about the element here.

Save & Exit button

Click here to save the changes and close this window.

Cancel button

This will abort the operation, not saving any changes.
