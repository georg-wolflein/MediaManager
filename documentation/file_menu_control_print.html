File menu control

Save Database

Click here to save the database to the specified location.

Refresh Database

This will reload the database from the specified location into the media database table.

View Raw Database...

This will open the raw database window, where you can view the raw CSV.

Print Selected Item(s)...

This will open a dialog allowing you to print stickers with media information that you can attach to your CDs. Select the items you want before printing.

Quit

Click here to exit the application.
